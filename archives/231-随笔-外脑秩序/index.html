唯一的分类维度

什么是维度？

  • 例如在生物界：界门纲目科属种的维度使每个生物都能放到准确的地方；
  • 维度就是你选择的角度要一致，分类后要求内容相互排斥，集体详尽无遗。

我的分类（展开）：

  • 第一级（按时间排列；先把项目这里，然后定期整理到第二级）

  • 近期归档（定期整理）

    • 收件箱（近期已经完成的）

    • 工作台（近期正在进行的）

      • Next Actions
      • Project
      • Waiting For
      • Someday/Maybe

  • 第二级（按性质排列；目前要使用的，便于检索查看）

  • 学习

    • 博学

      • 兴趣爱好
      • 基本常识

    • 专学

      • 课程
      • 实验

  • 生活

    • 衣食住行
    • 吃喝玩乐

  • 工作

    • 主业
    • 副业

  • 第三级（按需求排列；从第二级剔除的，防止第二级过于臃肿）

  • 存档（目前很久用不上的项目、废弃的项目）

第一级和第三级是服务性质的，是笔记管理的一部分。第二级是笔记的主要部分。

GTD: Getting Things Done

把需要做的事情处理好,GTD的五个核心原则是：收集、整理、组织、回顾、执行。

利用GTD，将人脑中关于需要完成的各种事情、任务移动出来而集中精力去做正在完成的事情。

这样使大脑可以不用塞满杂乱的任务，减少压力，提高工作效率。

收集

也就是你笔记中的000-归档。

第一级000-归档里面就是你一段时期中将要去做的所有事和已经完成的所有事。

整理

定期整理你的000-归档，争取把它们送入第二级中去。

具体来讲，整理你的000-归档必须遵循一个严格的流程，而且必须有时间限制~（花费过多时间不利于文件归档的体验，需要耗费较长时间的项目请使用专门的时间完成）~：

  • 从最上面开始；

  • 一次处理一项；

  • 着手后不把任何东西放回000-归档，必须经过以下某处理；

    • 第一，完成这件事（最多花两分钟）。
    • 第二，委托别人完成。
    • 第三，直接把它延期&将已完成部分归入第二级。
    • 第四，扔掉它（归入第三级）。

组织

你的文件目录包括了这一部分

回顾

利用该笔记管理系统，换句话说就是这个外脑，你必须及时回顾检查。回顾你的工作台，并且更新他们。

GTD要求你至少以星期为周期去回顾所有你比较主要的“行动”,“项目”和“等待”的事项，确保所有的新任务或者即将到来的事件都进入你的系统，而且所有的事情都更新到符合最新的情况。

执行

如果你把时间都花在组织工作上，而不是完成它们，那么所有的GTD系统都不会好用!

把要做的事情变得简单、有趣。

GTD 其实就是两点：

  • 清空你大脑的内存： 通常一件事情在你脑袋里所占的空间大小和你现实中已经完成多少成反比。把所有要做的事情分门别类，另外 存放在一个逻辑性强而又可靠的系统中去。大脑是用来思考的，不是用来记事的。我们的大脑每天耗费太多时间提醒我们该做又没做的事情，应该把大脑从这种思维中解放出来。
  • 把任何任务和项目具体成行动： 不管是什么任务项目，我们要问的问题是下一个动作是什么？只有这样才能把计划具体化，才能让我们随时把握该 做什么。这样一来，在一个给定时间具体采取一个给定行动（action）的时候，我们才能做出最好的选择，而且对自己的行动选择有信心。
