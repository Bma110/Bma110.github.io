Snippets

使用 Vscode Snippet 功能

编辑代码片段（Snippets）：Ctrl+Shift+p，搜索“snipets”，选择“配置用户代码片段”,选择“markdown.js”

使用自动补全（输入时在光标处的弹出窗口）：

  • Up、Down
  • Tab/Enter：确定

内置 Snippets

  • bold：粗体
  • italic：斜体
  • fenced codeblock：代码块
  • code：行内代码
  • fenced math：数学公式
  • inline math：行内数学公式
  • heading1 … heading6：1 到 6 级标题（小技巧：输入 h4 即可触发 heading4 的提示，其他类似）

一般格式

每一个代码片段格式为：

1
2
3
4
5
6
7
8

"rellink": {
  "prefix": "rrf",
  "description": "relative permalink",
  "body": [
    "[${1:name}]({{< relref \"${2:ref}\" >}})",
    "hello"
  ]
}, // 注意这个逗号
  • 每个代码片段均有一个唯一的标识符，如"rellink"
  • prefix: 触发文本，输入该文本（如rrf）
  • description：描述文本，随意
  • body：代码块主体，可以用一个字符串表示单行，或一个列表表示多行。
json 文件两项之间用逗号，最后一项后的逗号也不违反语法。因此，推荐每一项之后都加一个逗号。

占位符

使用 Tab 和 Shift+Tab 可以在占位符之间移动。
1

${<num>:<prefill text>}
  • <num>：移动顺序
  • <prefill text>：预填充文本

特殊占位符：

  • $<num>：复制 <num> 号占位符
  • $0：移动的终点，移动到此处时结束，不能再反向移动
